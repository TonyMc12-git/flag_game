# Flag Game

A touch-first country flag quiz for mobile.

The app presents one country flag and 20 country-name options. Tap an option to answer, see instant feedback, then continue to the next randomly generated round. The score keeps running for as long as you play.

## Local testing

Run a local static server from this folder:

```powershell
python -m http.server 8010
```

Then open:

```text
http://localhost:8010/index.html
```

## Phone install

After deployment to GitHub Pages, open the Pages URL on your phone and use `Add to Home Screen`.
